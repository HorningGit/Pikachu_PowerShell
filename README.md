## Pikachu PowerShell
___
<!-- Version -->
- Display Windows Version

```
$PSVersionTable
```
<!-- Execution Policy -->
- Display Execution Policy

```
$Get-ExecutionPolicy
```

###### Default Execution Policy by Version
Windows Version | Default Execution Policy
:------|:------:
Server 2019 | Remote Signed
Server 2016 | Remote Signed
Windows 10 | Restricted
